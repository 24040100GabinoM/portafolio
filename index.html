Bienvenido

Bienvenido al Portafolio de Evidencias, una aplicación web desarrollada para presentar de forma clara y estructurada los conocimientos adquiridos en la materia de Frameworks para Desarrollo Web. Este sitio reúne los principales procesos, conceptos y prácticas aplicadas durante el desarrollo de aplicaciones web modernas.

¿Qué aprenderas?

En este portafolio aprenderás los procesos fundamentales del desarrollo de aplicaciones web mediante el uso de frameworks modernos, integrando frontend, backend y principios de seguridad. El contenido está enfocado en la aplicación de buenas prácticas y metodologías utilizadas en entornos reales de desarrollo.

Objetivo del proyecto

El objetivo de este proyecto es desarrollar un portafolio de evidencias en forma de aplicación web que documente el aprendizaje adquirido en la materia de Frameworks para Desarrollo Web. A través de su estructura y contenidos, se busca demostrar el uso adecuado de herramientas, procesos y buenas prácticas del desarrollo web.

Desarrollo Frontend

Unidad 1: Proceso de Desarrollo de Frontend

Esta unidad aborda el diseño y construcción de interfaces web utilizando frameworks frontend, aplicando principios de diseño responsivo, experiencia de usuario y buenas prácticas de desarrollo.

Desarrollo Backend

Unidad 2: Proceso de Desarrollo Backend

En esta unidad se describen los procesos relacionados con la lógica del servidor, la gestión de datos y la comunicación entre el frontend y el backend mediante tecnologías web.

Seguridad e Implementación

Unidad 3: Proceso de Desarrollo de Seguridad e implementacion

Esta unidad se enfoca en la aplicación de medidas básicas de seguridad, así como en los procesos de implementación y despliegue de aplicaciones web en servicios de hosting.
